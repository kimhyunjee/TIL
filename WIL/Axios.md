# Axios

- 브라우저와 node.js에서 사용할 수 있는 Promise 기반 HTTP 클라이언트 라이브러리
- 백엔드랑 프론트엔드랑 통신을 쉽게하기 위해 Ajax와 더불어 사용한다.
- 자바스크립트에는 fetch api가 있지만,프레임워크에서 ajax를 구현할땐 axios를 쓰는 편
